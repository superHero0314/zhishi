# CloudBase 云托管 PHP 示例

参考文档：https://docs.cloudbase.net/run/dockerize-php.html

## 手工部署

在此目录下运行：

```sh
tcb framework deploy -e your-env-id
```

## 一键部署

点击下方按钮一键部署：
